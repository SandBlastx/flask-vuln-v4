🔍 Testing the Vulnerability

This application demonstrates how vulnerabilities in transitive dependencies can affect your app:

  • Direct dependency: Flask (declared in requirements.txt)
  • Transitive dependency: Jinja2 (comes with Flask)
  • Vulnerability surface: The |xmlattr filter in templates

Try submitting various attribute values to test how the filter handles special characters.
